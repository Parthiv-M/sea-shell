TREE						SEA SHELL COMMAND MANUAL							TREE

NAME
	tree -- Recursively display all files and directories stemming from the input directory

SYNOPSIS
	tree [DIRECTORY]
	tree [--help]

DESCRIPTION
	This manual page documents version 1.0.0 of the tree command.

	Recurses through all entries of the input, displays all files, and repeats the process for all subdirectories.

	If the input directory given is not valid, an error message is thrown.
