NAME
	dismantle -- remove a directory

SYNOPSIS
	dismantle PARENT_DIRECTORY DIRECTORY_NAME
	dismantle [--help]

DESCRIPTION
	This manual page documents version 1.0.0 of the dismantle command.

	Remove a directory in the PARENT_DIRECTORY with the name DIRECTORY_NAME, if they exist
