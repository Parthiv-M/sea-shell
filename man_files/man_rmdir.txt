NAME
	rmdir -- remove a directory

SYNOPSIS
	rmdir PARENT_DIRECTORY DIRECTORY_NAME
	rmdir [--help]

DESCRIPTION
	Remove a directory in the PARENT_DIRECTORY with the name DIRECTORY_NAME, if they exist
